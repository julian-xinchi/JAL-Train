## 五、电子邮件/URL 读音

在报邮箱或网址时常用：

| 符号 | 读法 | 示例 |
| :--- | :--- | :--- |
| `@` | "at" | `name@domain.com` → "name at domain dot com" |
| `.` | "dot" | `google.com` → "google dot com" |
| `_` | "underscore" | `user_name` → "user underscore name" |
| `-` | "hyphen" | `user-name` → "user hyphen name" |
| `/` | "slash" | `path/to/file` → "path slash to slash file" |
| `~` | "tilde" | `~user` → "tilde user" |
| `#` | "hash" | `#section` → "hash section" |
| `?` | "question mark" | `?id=1` → "question mark id equals 1" |
| `&` | "and" / "ampersand" | `a&b` → "a and b" |

## 💡 总结

- **日常打字**：说中文俗称就够了（如"at"、"井号"）
- **报邮箱/网址**：用 "at"、"dot"、"underscore"、"hyphen"
- **编程交流**：用英文原名（如 "backtick"、"pipe"、"slash"）
- **数学场合**：使用标准数学读法（如 "plus"、"minus"、"times"）
